# Mini JavaScript Interpreter

A mini JavaScript interpreter built in C++ for Thunder Hackathon.

## Supported Features

* Odd / Even Checker
* Triangle Pattern Generator
* Armstrong Number Checker
* Array Reverse
* Palindrome Checker

## Compile

```bash
g++ compiler_final.cpp -o compiler
```

## Run

```bash
./compiler
```

Paste JavaScript code and press:

Windows: Ctrl + Z, then Enter
